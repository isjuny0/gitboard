# 📘 GitBoard
Spring Boot로 만든 간단한 게시판 API입니다.

## 📌 프로젝트 소개
Git 연습을 겸한 Spring Boot 기반 게시판 API입니다.  
게시글 등록, 조회, 삭제 기능을 제공합니다.

## 🚀 기술 스택
- Java 17
- Spring Boot 3.x
- Spring Data JPA
- H2 Database (개발용)
- Git, GitHub

## 🛠️ 기능
- 게시글 등록 `POST /boards`
- 게시글 목록 조회 `GET /boards`
- 게시글 삭제 `DELETE /boards/{id}`
